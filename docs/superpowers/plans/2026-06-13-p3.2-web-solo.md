# P3.2 — Solo client-local web : plan d'implémentation

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Câbler `apps/web` (TanStack Start) pour offrir une partie solo complète et jouable au navigateur contre l'IA, en client-local `boardgame.io`, sans réseau ni persistance.

**Architecture:** App shell TanStack Start réutilisable (root layout + routing file-based). Le match solo utilise **deux clients `boardgame.io` vanilla** partageant une même instance `Local()` et un `matchID` : un client humain (`'0'`) bridgé vers React via souscription, et un client IA (`'1'`) piloté par un driver qui appelle les fonctions pures `chooseSideDeck`/`chooseMove` d'`engine` (Approche A). Aucune règle dupliquée côté web : `engine` reste la seule source de vérité.

**Tech Stack:** TanStack Start 1.168.25 + TanStack Router 1.170.15, React 19.2.7, Vite, `boardgame.io` 0.50.2 (client vanilla `boardgame.io/client` + `Local` de `boardgame.io/multiplayer`), Zustand 5.0.14, Vitest 4.1.8.

**Spec de référence :** `docs/superpowers/specs/2026-06-13-p3.2-web-solo-design.md`

- [ ] **Step 1: Écrire le test `test/ai-driver.test.ts`**

On pilote **les deux** sièges par `aiStep` : l'IA contre elle-même est une partie valide de bout en bout. On boucle un pas sur le client dont c'est le tour jusqu'à `matchWinner`, avec un garde-fou d'itérations.

```ts
import { expect, test } from "vitest";
import { createSoloClients } from "../src/solo/clients";
import { aiStep } from "../src/solo/ai-driver";

test("deux sièges pilotés par l'IA jouent jusqu'à un vainqueur", () => {
  const { human, ai } = createSoloClients("ai-vs-ai");
  human.start();
  ai.start();

  const params = { standThreshold: 18 } as const;
  let guard = 0;
  // Phase pick : chaque client pose son side deck quand il est actif.
  // Phase play : le client dont c'est le tour avance d'un pas.
  while (human.getState()?.G.matchWinner == null && guard < 5000) {
    guard++;
    const acted =
      aiStep(human.getState(), human.moves, params) ||
      aiStep(ai.getState(), ai.moves, params);
    if (!acted) break; // personne ne peut agir : sécurité anti-boucle
  }

  const final = human.getState();
  expect(final?.G.matchWinner).not.toBeNull();
  expect(["0", "1"]).toContain(final?.G.matchWinner);
  // Le vainqueur a 3 sets (RULES §6). `players['1']` est strippé côté client humain
  // mais setsWon est public.
  const winner = final?.G.matchWinner;
  expect(final?.G.players[winner!]?.setsWon).toBe(3);

  human.stop();
  ai.stop();
});
```

- [ ] **Step 3: Écrire `src/solo/ai-driver.ts`**

`aiStep` est pur vis-à-vis du framework : il lit l'état (déjà strippé pour le siège du client → le joueur voit sa propre main complète), décide via `engine`, dispatche **au plus un** coup, et renvoie `true` s'il a agi. Il ne fait rien si l'état est en cours de sync ou si le client n'est pas actif.

```ts
import { chooseMove, chooseSideDeck } from "@pazaak/engine";
import type { AiParams, PlayerState } from "@pazaak/engine";

/** État renvoyé par client.getState() (sous-ensemble utilisé). */
interface ClientState {
  G: {
    players: Record<string, PlayerState>;
    matchWinner: string | null;
  };
  ctx: { phase: string | null; currentPlayer: string };
  isActive?: boolean;
}

interface ClientMoves {
  [move: string]: (...args: unknown[]) => void;
}

/**
 * Avance le pilotage IA d'un pas pour le client donné. Renvoie true si un coup a été
 * dispatché. Sans effet si l'état est nul (sync) ou si le client n'est pas actif.
 * - pickSideDeck : pose le side deck par défaut de l'IA si pas encore fait ;
 * - play : applique chooseMove (rescousse / stand / jeu vers 20 / endTurn).
 */
export function aiStep(
  state: ClientState | null,
  moves: ClientMoves,
  params: AiParams,
): boolean {
  if (!state || state.isActive === false) {
    return false;
  }
  const self = state.G.players[state.ctx.currentPlayer];
  if (!self) {
    // Phase pick : currentPlayer peut ne pas désigner ce client ; on retrouve le joueur
    // non encore pické parmi ceux dont on a la vue complète (hand est un tableau).
    if (state.ctx.phase === "pickSideDeck") {
      for (const p of Object.values(state.G.players)) {
        if (Array.isArray(p.hand) && p.sideDeck === null) {
          moves.pickSideDeck?.(chooseSideDeck());
          return true;
        }
      }
    }
    return false;
  }

  if (state.ctx.phase === "pickSideDeck") {
    if (self.sideDeck === null) {
      moves.pickSideDeck?.(chooseSideDeck());
      return true;
    }
    return false;
  }

  // Phase play : l'IA est active et c'est son tour.
  const m = chooseMove(self, params);
  moves[m.move]?.(...m.args);
  return true;
}
```

> Note d'implémentation : en phase `pickSideDeck`, `ctx.currentPlayer` n'est pas fiable (jeu simultané `activePlayers`). On identifie « le joueur de ce client non encore pické » par le fait que sa `hand` est un tableau (vue complète) et `sideDeck === null`. La vue adverse a `hand: { count }` (objet, pas tableau) → jamais confondue. Si la résolution Context7 révèle que `getState()` expose un identifiant de siège plus direct (ex. `ctx.playerID` côté stage), préférer ce signal et simplifier.

```ts
export type Screen = "pick" | "play" | "gameover";

interface ScreenState {
  G: { matchWinner: string | null };
  ctx: { phase: string | null; gameover?: unknown };
}

/** Écran à afficher, dérivé de l'état du client humain. */
export function deriveScreen(state: ScreenState | null): Screen {
  if (!state) {
    return "pick";
  }
  if (state.G.matchWinner != null || state.ctx.gameover != null) {
    return "gameover";
  }
  if (state.ctx.phase === "pickSideDeck") {
    return "pick";
  }
  return "play";
}
```

```ts
export const DECK_SIZE = 10;

/**
 * Bascule l'index `i` dans la sélection : retire s'il y est, ajoute sinon (sauf si la
 * sélection est déjà pleine à DECK_SIZE → no-op). La désélection reste toujours possible.
 */
export function toggleSelection(selected: number[], i: number): number[] {
  if (selected.includes(i)) {
    return selected.filter((x) => x !== i);
  }
  if (selected.length >= DECK_SIZE) {
    return selected;
  }
  return [...selected, i];
}
```

```tsx
import type { PlayedCard, SideCard } from "@pazaak/engine";

/** Libellé d'une carte de side deck (catalogue). */
export function sideCardLabel(card: SideCard): string {
  if (card.kind !== "standard") {
    return "?"; // gold cards : P8
  }
  const prefix = card.sign === "pm" ? "±" : card.sign;
  return `${prefix}${card.value}`;
}

/** Libellé d'une carte posée sur le plateau. */
export function playedCardLabel(card: PlayedCard): string {
  if (card.source === "main") {
    return `${card.value}`;
  }
  return card.resolvedValue >= 0 ? `+${card.resolvedValue}` : `${card.resolvedValue}`;
}

  return (
    <span

        padding: "0.5rem",
        margin: "0.15rem",
        border: "1px solid #888",
        borderRadius: "4px",
        textAlign: "center",

      {label}

}
```

```ts
import { useEffect, useState } from "react";
import type { SoloClient } from "./clients";

/** État courant exposé par un client vanilla (getState()). */
export type SoloState = ReturnType<SoloClient["getState"]>;

/** Souscrit au client humain et renvoie son état courant (re-render à chaque changement). */
export function useSoloGame(client: SoloClient): SoloState {
  const [state, setState] = useState<SoloState>(() => client.getState());
  useEffect(() => {
    setState(client.getState());
    const unsubscribe = client.subscribe(() => setState(client.getState()));
    return unsubscribe;
  }, [client]);
  return state;
}
```

- [ ] **Step 2: Écrire `src/solo/use-ai-driver.ts`**

Le client IA est piloté à chaque notification. On dispatche via `queueMicrotask` + un flag de ré-entrance pour éviter la récursion synchrone du transport `Local` (un `moves.x()` notifie immédiatement les abonnés).

```ts
import { useEffect } from "react";
import type { AiParams } from "@pazaak/engine";
import { aiStep } from "./ai-driver";
import type { SoloClient } from "./clients";

/** Branche le pilotage IA sur le client du siège '1' pour toute la durée du match. */
export function useAiDriver(client: SoloClient, params: AiParams): void {
  useEffect(() => {
    let acting = false;
    const pump = () => {
      if (acting) {
        return;
      }
      acting = true;
      queueMicrotask(() => {
        acting = false;
        // aiStep ne dispatche qu'au plus un coup et re-vérifie l'état : sûr à rappeler.
        aiStep(client.getState(), client.moves, params);
      });
    };
    const unsubscribe = client.subscribe(pump);
    pump(); // amorce le pick initial de l'IA
    return unsubscribe;
  }, [client, params]);
}
```

- [ ] **Step 2: Orchestration `src/routes/solo.tsx`**

Crée les clients une fois (mémoïsés sur la difficulté), branche les hooks, et rend l'écran dérivé. Le `matchID` change à chaque « Rejouer » pour repartir d'un master neuf.

```tsx
import { createFileRoute } from "@tanstack/react-router";
import { useMemo, useState } from "react";
import { aiParamsFor, isDifficulty } from "../solo/difficulty";
import { createSoloClients } from "../solo/clients";
import { deriveScreen } from "../solo/screen";
import { useAiDriver } from "../solo/use-ai-driver";
import { useSoloGame } from "../solo/use-solo-game";
import { useDeckBuilder } from "../store";
import { Board } from "../components/Board";
import { DeckBuilder } from "../components/DeckBuilder";
import { GameOver } from "../components/GameOver";

export const Route = createFileRoute("/solo")({
  validateSearch: (search): { difficulty: "easy" | "normal" | "hard" } => ({
    difficulty: isDifficulty(search.difficulty) ? search.difficulty : "normal",
  }),
  component: SoloPage,
});

function SoloPage() {
  const { difficulty } = Route.useSearch();
  const [matchSeq, setMatchSeq] = useState(0);
  const params = useMemo(() => aiParamsFor(difficulty), [difficulty]);
  const { human, ai } = useMemo(() => {
    const clients = createSoloClients(`solo-${difficulty}-${matchSeq}`);
    clients.human.start();
    clients.ai.start();
    return clients;
  }, [difficulty, matchSeq]);

  useAiDriver(ai, params);
  const state = useSoloGame(human);
  const resetDeck = useDeckBuilder((s) => s.reset);

  const screen = deriveScreen(state);

  if (screen === "pick") {
    return (
      <DeckBuilder
        onConfirm={(deck) => {
          human.moves.pickSideDeck(deck);
        }}
      />
    );
  }

  if (screen === "gameover" && state) {
    const winner = state.G.matchWinner ?? "0";
    return (
      <GameOver
        winner={winner}
        isHumanWinner={winner === "0"}
        onReplay={() => {
          resetDeck();
          setMatchSeq((n) => n + 1);
        }}
      />
    );
  }

  if (state && screen === "play") {
    const human0 = state.G.players["0"];
    const ai1 = state.G.players["1"];
    if (!human0 || !ai1) {
      return <div data-testid="solo-loading">Chargement…</div>;
    }
    return (
      <Board
        human={human0}
        opponent={ai1}
        isMyTurn={state.ctx.currentPlayer === "0" && state.isActive === true}
        currentSet={state.G.currentSet}
        onPlay={(i, decl) => human.moves.playCard(i, decl)}
        onEndTurn={() => human.moves.endTurn()}
        onStand={() => human.moves.stand()}
      />
    );
  }

  return <div data-testid="solo-loading">Chargement…</div>;
}
```
